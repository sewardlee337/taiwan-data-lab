Taiwan Data Lab is built upon the shoulders of free and/or open source technologies, including:

  • Flask - Python web framework
  • Chart.js - JavaScript plug-in for data visualization
  • Leaflet - JavaScript library for interactive maps
  • D3.js - JavaScript library for data visualization using HTML, SVG, and CSS
  • DataTables - Table plug-in for jQuery
  • jQuery Sparklines - jQuery plugin that generates sparklines
  • SQLite - Embedded SQL database engine
  • Bootstrap - HTML, CSS, JavaScript framework for responsive webpages
